## ⚙️ Optional Enhancements

### Add Your Photo
```html
<img src="your-photo.jpg" alt="Your Name" style="border-radius: 15px; max-width: 300px;">
```

### Add Project Images
Replace icons with actual screenshots:
```html
<img src="project-screenshot.png" alt="Project Name">
```

### Change Colors
Update in `styles.css`:
```css
--primary-color: #your-color-here;
--secondary-color: #your-color-here;
```

### Enable Email Form
Replace `script.js` contact form with Formspree:
1. Go to formspree.io
2. Create new form
3. Update action URL in HTML

## 📞 Support

### Common Issues & Solutions

**Images not showing?**
- Check file paths are relative
- Use correct image formats (PNG, JPG)

**Colors look different?**
- Clear browser cache (Ctrl+Shift+Del)
- Try different browser

**Mobile menu not working?**
- Check browser console for errors
- Ensure JavaScript is enabled

**Links not working?**
- Verify URLs are correct
- Use full URLs for external links
