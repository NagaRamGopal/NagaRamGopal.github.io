# 🚀 Naga Ram Gopal - Professional Portfolio

A stunning, modern portfolio website designed to impress recruiters and showcase your professional work. Built with vanilla HTML, CSS, and JavaScript for optimal performance.

## ✨ Features

- **Dark Theme** - Eye-catching dark background with vibrant gradient accents
- **Responsive Design** - Fully responsive and mobile-optimized
- **Smooth Animations** - Elegant transitions and interactive elements
- **Performance Optimized** - Lightweight and fast-loading
- **Modern UI/UX** - Clean, professional design that stands out
- **Interactive Sections**:
  - Hero section with floating cards
  - About section with stats and code showcase
  - Skills categorized by expertise
  - Featured projects with hover effects
  - Professional experience timeline
  - Contact form and social links
  - Scroll-to-top button

## 🚀 Getting Started

1. **Clone or Download** this repository
2. **Open** `index.html` in your web browser
3. **Customize** the content with your information:
   - Update your name, email, and location
   - Add your actual projects
   - Update skills and experience
   - Add links to your social profiles

## 🔧 Customization Guide

### Personal Information
Edit the following sections in `index.html`:
- Hero section title and subtitle
- About section text
- Contact information (email, phone, location)
- Social media links

### Projects
Update the project cards with:
- Project titles and descriptions
- Technologies used
- Links to live demos and GitHub repositories
- Replace placeholder icons with project screenshots

### Skills
Modify the skills categories:
- Add/remove skill tags
- Update categories as needed

### Experience
Update the timeline with your actual experience:
- Job titles and companies
- Dates
- Achievements

### Colors
To change the color scheme, edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Change this */
    --secondary-color: #ff006e;    /* Change this */
    /* ... other colors ... */
}
```

## 🚀 Deployment Options

Deploy your portfolio to:
- **GitHub Pages** - Free hosting from GitHub
- **Netlify** - Drag and drop deployment
- **Vercel** - Optimized for static sites
- **Traditional Hosting** - Upload via FTP

### Deploy to GitHub Pages:
1. Create a GitHub repository named `username.github.io`
2. Push this portfolio files
3. Enable GitHub Pages in repository settings
4. Your portfolio will be live at `https://username.github.io`
